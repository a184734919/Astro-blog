# 前端错误处理

## 概述

前端错误处理是现代 Web 开发中的重要组成部分。随着应用复杂度提升，错误来源也变得多样化，例如：

- JavaScript 运行时错误
- Promise 异步错误
- 网络请求错误
- 资源加载错误
- 框架生命周期错误
- 业务逻辑错误

完善的错误处理机制可以提升系统稳定性与用户体验。

---

## 为什么需要错误处理

缺少统一错误处理会导致：

- 页面白屏
- 功能异常不可用
- 用户体验下降
- 线上问题难以排查
- 无法进行错误统计与监控

因此通常需要构建完整的错误处理体系：

- 错误捕获
- 错误分类
- 错误上报
- 用户提示
- 日志记录

## 错误类型

### 1. 同步错误

```javascript
try {
  const obj = undefined;
  console.log(obj.name);
} catch (err) {
  console.error("同步错误：", err);
  // 错误对象信息
  console.log(err.message);    // "Cannot read properties of undefined"
  console.log(err.stack);      // 堆栈信息
  console.log(err.name);       // "TypeError"
}

// 常见错误类型
// TypeError: 类型错误
// ReferenceError: 引用错误
// RangeError: 范围错误
// SyntaxError: 语法错误
// Error: 通用错误
```

### 2. 异步错误

```javascript
// Promise 错误
const promise = new Promise((resolve, reject) => {
  reject(new Error("Promise 错误"));
});

promise.catch(error => {
  console.error("Promise 错误：", error);
});

// async/await 错误
async function fetchData() {
  try {
    const response = await fetch("/api/data");
    return await response.json();
  } catch (error) {
    console.error("异步请求错误：", error);
    // 处理错误
    throw error; // 重新抛出或返回默认值
  }
}

// 全局 Promise 错误捕获
window.addEventListener('unhandledrejection', event => {
  console.error("未处理的 Promise 拒绝：", event.reason);
  // 阻止默认的控制台错误输出
  event.preventDefault();
});
```

### 3. 网络请求错误

```javascript
// Fetch API 错误处理
async function fetchWithErrorHandling(url) {
  try {
    const response = await fetch(url);

    if (!response.ok) {
      // HTTP 错误状态码
      throw new Error(`HTTP 错误：${response.status} ${response.statusText}`);
    }

    return await response.json();
  } catch (error) {
    // 网络错误或解析错误
    console.error("请求失败：", error);
    // 返回默认值或重试
    return null;
  }
}

// Axios 错误处理
import axios from 'axios';

const api = axios.create({
  baseURL: '/api',
  timeout: 10000
});

// 请求拦截器
api.interceptors.request.use(
  config => {
    // 添加认证 token
    const token = localStorage.getItem('token');
    if (token) {
      config.headers.Authorization = `Bearer ${token}`;
    }
    return config;
  },
  error => {
    return Promise.reject(error);
  }
);

// 响应拦截器
api.interceptors.response.use(
  response => {
    return response.data;
  },
  error => {
    if (error.response) {
      // 服务器返回了错误状态码
      switch (error.response.status) {
        case 400:
          console.error('请求参数错误');
          break;
        case 401:
          console.error('未授权，请登录');
          // 跳转到登录页
          window.location.href = '/login';
          break;
        case 403:
          console.error('拒绝访问');
          break;
        case 404:
          console.error('请求资源不存在');
          break;
        case 500:
          console.error('服务器内部错误');
          break;
        default:
          console.error('未知错误');
      }
    } else if (error.request) {
      // 请求已发出但没有收到响应
      console.error('网络错误：', error.message);
    } else {
      // 请求配置错误
      console.error('请求配置错误：', error.message);
    }

    return Promise.reject(error);
  }
);
```

```javascript
// 图片加载错误处理
function handleImageError(imgElement, fallbackSrc) {
  imgElement.onerror = function() {
    console.error('图片加载失败');
    imgElement.src = fallbackSrc;
    imgElement.alt = '图片加载失败';

handleImageError(img, '/assets/placeholder.png');

window.addEventListener('error', event => {
  if (event.target !== window) {
    // 资源加载错误
    console.error('资源加载失败：', event.target.src);
    event.preventDefault();
  }
}, true);

// Script 加载错误处理
function loadScript(src, callback) {
  const script = document.createElement('script');
  script.src = src;
  script.onload = callback;
  script.onerror = () => {
    console.error(`脚本加载失败：${src}`);
    callback(new Error(`脚本加载失败：${src}`));
  };
  document.head.appendChild(script);
}
```

## 全局错误处理

### 1. 全局错误捕获

```javascript
// 全局错误处理器
class GlobalErrorHandler {
  constructor() {
    this.init();
  }

  init() {
    // 捕获同步错误
    window.addEventListener('error', this.handleGlobalError.bind(this));

    // 捕获 Promise 错误
    window.addEventListener('unhandledrejection', this.handleUnhandledRejection.bind(this));

    // 捕获资源加载错误
    window.addEventListener('error', this.handleResourceError.bind(this), true);
  }

  // 处理全局错误
  handleGlobalError(event) {
    const { message, filename, lineno, colno, error } = event;

    console.error('全局错误：', {
      message,
      filename,
      lineno,
      colno,
      error
    });

    // 上报错误
    this.reportError({
      type: 'global',
      message,
      filename,
      lineno,
      colno,
      stack: error?.stack
    });
  }

  // 处理未处理的 Promise 拒绝
  handleUnhandledRejection(event) {
    const error = event.reason;

    console.error('未处理的 Promise 拒绝：', error);

    // 上报错误
    this.reportError({
      type: 'promise',
      message: error.message,
      stack: error.stack
    });

    // 阻止默认错误输出
    event.preventDefault();
  }

  // 处理资源加载错误
  handleResourceError(event) {
    if (event.target !== window) {
      const target = event.target;
      console.error('资源加载失败：', {
        tagName: target.tagName,
        src: target.src || target.href
      });

      // 上报错误
      this.reportError({
        type: 'resource',
        tagName: target.tagName,
        src: target.src || target.href
      });
    }
  }

  // 上报错误
  async reportError(errorInfo) {
    try {
      await fetch('/api/error-report', {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json'
        },
        body: JSON.stringify({
          ...errorInfo,
          userAgent: navigator.userAgent,
          url: window.location.href,
          timestamp: new Date().toISOString()
        })
      });
    } catch (error) {
      console.error('错误上报失败：', error);
    }
  }
}

// 初始化全局错误处理器
const errorHandler = new GlobalErrorHandler();
```

### 2. React 错误边界

```javascript
import React from 'react';

class ErrorBoundary extends React.Component {
  constructor(props) {
    super(props);
    this.state = {
      hasError: false,
      error: null,
      errorInfo: null
    };
  }

  static getDerivedStateFromError(error) {
    // 更新 state 使下一次渲染显示降级 UI
    return { hasError: true };
  }

  componentDidCatch(error, errorInfo) {
    // 可以将错误日志上报给服务器
    console.error('React 错误边界捕获的错误：', error, errorInfo);

    this.setState({
      error,
      errorInfo
    });

    // 上报错误
    this.reportError(error, errorInfo);
  }

  async reportError(error, errorInfo) {
    try {
      await fetch('/api/error-report', {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json'
        },
        body: JSON.stringify({
          error: error.message,
          stack: error.stack,
          componentStack: errorInfo.componentStack,
          userAgent: navigator.userAgent,
          url: window.location.href
        })
      });
    } catch (reportError) {
      console.error('错误上报失败：', reportError);
    }
  }

  handleReset = () => {
    this.setState({
      hasError: false,
      error: null,
      errorInfo: null
    });
  };

  render() {
    if (this.state.hasError) {
      // 自定义错误 UI
      return (
        <div className="error-boundary">
          <h1>出错了</h1>
          <p>抱歉，应用程序遇到了一个错误。</p>
          {this.state.error && (
            <details>
              <summary>错误详情</summary>
              <pre>{this.state.error.toString()}</pre>
              <pre>{this.state.errorInfo?.componentStack}</pre>
            </details>
          )}
          <button onClick={this.handleReset}>重试</button>
          <button onClick={() => window.location.reload()}>刷新页面</button>
        </div>
      );
    }

    return this.props.children;
  }
}

// 使用示例
function App() {
  return (
    <ErrorBoundary>
      <MainComponent />
    </ErrorBoundary>
  );
}
```

## 实际应用

### 1. 错误监控和上报系统

```javascript
// 错误监控系统
class ErrorMonitor {
  constructor(config = {}) {
    this.config = {
      reportUrl: '/api/error-report',
      maxErrors: 100,
      reportInterval: 5000,
      environment: 'production',
      ...config
    };

    this.errorQueue = [];
    this.isReporting = false;
    this.userId = this.getUserId();
    this.sessionId = this.getSessionId();
  }

  init() {
    // 初始化各种错误监听
    this.setupGlobalErrorHandler();
    this.setupPromiseErrorHandler();
    this.setupResourceErrorHandler();
    this.setupPerformanceObserver();
    this.startPeriodicReport();
  }

  setupGlobalErrorHandler() {
    window.addEventListener('error', (event) => {
      this.captureError({
        type: 'javascript',
        message: event.message,
        filename: event.filename,
        lineno: event.lineno,
        colno: event.colno,
        stack: event.error?.stack
      });
    });
  }

  setupPromiseErrorHandler() {
    window.addEventListener('unhandledrejection', (event) => {
      this.captureError({
        type: 'promise',
        message: event.reason?.message,
        stack: event.reason?.stack
      });
    });
  }

  setupResourceErrorHandler() {
    window.addEventListener('error', (event) => {
      if (event.target !== window) {
        this.captureError({
          type: 'resource',
          tagName: event.target.tagName,
          src: event.target.src || event.target.href
        });
      }
    }, true);
  }

  setupPerformanceObserver() {
    if ('PerformanceObserver' in window) {
      const observer = new PerformanceObserver((list) => {
        for (const entry of list.getEntries()) {
          if (entry.entryType === 'error') {
            this.captureError({
              type: 'performance',
              message: entry.message,
              stack: entry.stack
            });
          }
        }
      });
      observer.observe({ entryTypes: ['error'] });
    }
  }

  captureError(errorInfo) {
    const enrichedError = {
      ...errorInfo,
      userId: this.userId,
      sessionId: this.sessionId,
      url: window.location.href,
      userAgent: navigator.userAgent,
      timestamp: new Date().toISOString(),
      environment: this.config.environment
    };

    // 添加到队列
    this.errorQueue.push(enrichedError);

    // 限制队列大小
    if (this.errorQueue.length > this.config.maxErrors) {
      this.errorQueue.shift();
    }

    // 如果达到批量上报阈值，立即上报
    if (this.errorQueue.length >= 10) {
      this.reportErrors();
    }
  }

  async reportErrors() {
    if (this.isReporting || this.errorQueue.length === 0) {
      return;
    }

    this.isReporting = true;

    try {
      const errors = [...this.errorQueue];
      this.errorQueue = [];

      await fetch(this.config.reportUrl, {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json'
        },
        body: JSON.stringify({ errors })
      });

      console.log(`成功上报 ${errors.length} 个错误`);
    } catch (error) {
      console.error('错误上报失败：', error);
      // 上报失败，重新加入队列
      this.errorQueue.unshift(...errors);
    } finally {
      this.isReporting = false;
    }
  }

  startPeriodicReport() {
    setInterval(() => {
      this.reportErrors();
    }, this.config.reportInterval);
  }

  getUserId() {
    // 从 localStorage 或 cookie 获取用户 ID
    return localStorage.getItem('userId') || 'anonymous';
  }

  getSessionId() {
    // 生成或获取会话 ID
    let sessionId = sessionStorage.getItem('sessionId');
    if (!sessionId) {
      sessionId = this.generateSessionId();
      sessionStorage.setItem('sessionId', sessionId);
    }
    return sessionId;
  }

  generateSessionId() {
    return `${Date.now()}-${Math.random().toString(36).substr(2, 9)}`;
  }

  // 手动捕获错误
  captureException(error, context = {}) {
    this.captureError({
      type: 'exception',
      message: error.message,
      stack: error.stack,
      ...context
    });
  }

  // 捕获自定义错误
  captureCustomError(message, level = 'error', context = {}) {
    this.captureError({
      type: 'custom',
      message,
      level,
      ...context
    });
  }
}

// 使用示例
const errorMonitor = new ErrorMonitor({
  reportUrl: 'https://your-api.com/errors',
  environment: process.env.NODE_ENV
});

errorMonitor.init();

// 手动捕获错误
try {
  // 一些可能出错的代码
} catch (error) {
  errorMonitor.captureException(error, {
    component: 'UserForm',
    action: 'submit'
  });
}
```

### 2. 错误重试机制

```javascript
// 错误重试工具
class RetryHandler {
  constructor(options = {}) {
    this.options = {
      maxRetries: 3,
      retryDelay: 1000,
      backoffFactor: 2,
      shouldRetry: this.defaultShouldRetry,
      onRetry: () => {},
      ...options
    };
  }

  defaultShouldRetry(error) {
    // 默认重试条件：网络错误或 5xx 错误
    return (
      error.message?.includes('Network Error') ||
      error.response?.status >= 500
    );
  }

  async execute(fn, context = {}) {
    let lastError;

    for (let attempt = 0; attempt <= this.options.maxRetries; attempt++) {
      try {
        const result = await fn(attempt);
        return result;
      } catch (error) {
        lastError = error;

        // 检查是否应该重试
        if (!this.options.shouldRetry(error, attempt, context)) {
          throw error;
        }

        // 如果还有重试次数，等待后重试
        if (attempt < this.options.maxRetries) {
          const delay = this.calculateDelay(attempt);
          console.log(`重试 ${attempt + 1}/${this.options.maxRetries}，延迟 ${delay}ms`);

          // 调用重试回调
          this.options.onRetry(error, attempt + 1, delay, context);

          await this.delay(delay);
        }
      }
    }

    // 所有重试都失败
    throw new Error(
      `操作失败，已重试 ${this.options.maxRetries} 次。最后错误：${lastError.message}`
    );
  }

  calculateDelay(attempt) {
    return this.options.retryDelay * Math.pow(this.options.backoffFactor, attempt);
  }

  delay(ms) {
    return new Promise(resolve => setTimeout(resolve, ms));
  }
}

// 使用示例
const retryHandler = new RetryHandler({
  maxRetries: 3,
  retryDelay: 1000,
  backoffFactor: 2,
  shouldRetry: (error, attempt) => {
    // 自定义重试条件
    return attempt < 3 && (error.response?.status >= 500 || error.message?.includes('Network'));
  },
  onRetry: (error, attempt, delay, context) => {
    console.log(`重试第 ${attempt} 次，延迟 ${delay}ms`);
    // 可以显示重试提示
    showToast(`网络异常，正在重试 (${attempt}/3)...`);
  }
});

// 带重试的 API 请求
async function fetchWithRetry(url, options = {}) {
  return retryHandler.execute(async () => {
    const response = await fetch(url, options);
    if (!response.ok) {
      throw new Error(`HTTP ${response.status}: ${response.statusText}`);
    }
    return response.json();
  }, { url, options });
}

// 使用
try {
  const data = await fetchWithRetry('/api/data');
  console.log('数据获取成功：', data);
} catch (error) {
  console.error('最终失败：', error);
  showError('数据加载失败，请稍后重试');
}
```

### 3. 错误恢复和降级策略

```javascript
// 错误恢复策略
class ErrorRecovery {
  constructor() {
    this.strategies = new Map();
  }

  // 注册恢复策略
  registerStrategy(errorType, strategy) {
    this.strategies.set(errorType, strategy);
  }

  // 执行恢复策略
  async recover(error, context = {}) {
    const errorType = this.getErrorType(error);
    const strategy = this.strategies.get(errorType);

    if (strategy) {
      try {
        return await strategy(error, context);
      } catch (recoveryError) {
        console.error('恢复策略执行失败：', recoveryError);
        return this.fallbackRecovery(error, context);
      }
    }

    return this.fallbackRecovery(error, context);
  }

  getErrorType(error) {
    if (error.response?.status === 401) return 'auth';
    if (error.response?.status === 403) return 'permission';
    if (error.response?.status >= 500) return 'server';
    if (error.message?.includes('Network')) return 'network';
    return 'unknown';
  }

  async fallbackRecovery(error, context) {
    console.error('使用默认恢复策略：', error);
    // 默认恢复策略
    return { success: false, message: '操作失败，请稍后重试' };
  }
}
```

```javascript
// 错误日志系统
class ErrorLogger {
  constructor() {
    this.logs = [];
    this.maxLogs = 100;
    this.storageKey = 'error-logs';
  }

  // 记录错误
  log(error, context = {}) {
    const logEntry = {
      timestamp: new Date().toISOString(),
      error: {
        message: error.message,
        stack: error.stack,
        name: error.name
      },
      context,
      url: window.location.href,
      userAgent: navigator.userAgent
    };

    this.logs.push(logEntry);

    // 限制日志数量
    if (this.logs.length > this.maxLogs) {
      this.logs.shift();
    }

    // 保存到本地存储
    this.saveLogs();
  }

  // 保存日志到本地存储
  saveLogs() {
    try {
      localStorage.setItem(this.storageKey, JSON.stringify(this.logs));
    } catch (error) {
      console.error('保存错误日志失败：', error);
    }
  }

  // 加载日志
  loadLogs() {
    try {
      const saved = localStorage.getItem(this.storageKey);
      if (saved) {
        this.logs = JSON.parse(saved);
      }
    } catch (error) {
      console.error('加载错误日志失败：', error);
    }
  }

  // 获取所有日志
  getLogs() {
    return [...this.logs];
  }

  // 清除日志
  clearLogs() {
    this.logs = [];
    localStorage.removeItem(this.storageKey);
  }

  // 导出日志
  exportLogs() {
    const data = JSON.stringify(this.logs, null, 2);
    const blob = new Blob([data], { type: 'application/json' });
    const url = URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = `error-logs-${Date.now()}.json`;
    a.click();
    URL.revokeObjectURL(url);
  }
}
```

## 总结

前端错误处理是一个系统性的工程，需要从多个层面进行考虑：

### 核心要点

1. **错误分类**：同步错误、异步错误、网络错误、资源错误
2. **错误捕获**：全局捕获、框架级捕获、组件级捕获
3. **错误处理**：错误恢复、重试机制、降级策略
4. **错误上报**：错误监控、日志记录、数据分析
5. **用户体验**：友好提示、离线支持、优雅降级

### 最佳实践

- 建立完整的错误处理体系
- 实现智能的错误恢复机制
- 提供友好的用户提示
- 做好错误监控和分析
- 注意性能和安全性

### 工具推荐

- Sentry: 专业的错误监控平台
- Rollbar: 实时错误追踪和报告
- Bugsnag: 错误监控和性能分析
- 自研错误监控系统：根据业务需求定制

通过完善的错误处理机制，可以显著提升应用的稳定性和用户体验。记住错误处理不是简单的 try-catch，而是一个完整的系统设计。
